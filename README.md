# Jaco van der Merwe Portfolio

A professional executive-style portfolio website for Jaco van der Merwe, National Parts Manager.

## Purpose

This portfolio presents Jaco's automotive parts leadership experience, award recognition, career history, core expertise, and contact details in a clean, recruiter-friendly format.

## Features

- Responsive single-page portfolio
- Executive hero section
- Achievement highlight
- Core expertise cards
- Current role focus cards
- Career timeline
- Referee section
- Downloadable CV link
- GitHub Pages friendly static build

## How to Run Locally

Open `index.html` in your browser.

## How to Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to repository Settings.
4. Open Pages.
5. Select branch `main` and root folder `/`.
6. Save and wait for GitHub Pages to publish the site.
